
USAGE: {} [OPTS] <input.kdl> [output.html]

FLAGS:
        -h, --help
                Shows this menu and quits.

	-m, --minify
		Disables pretty mode. Essentially uses the least spacing and indentation possible.
		Awesome for saving space.

	-t, --tab-size=[number]
		Allows you to configure the indentation depth, using space characters.
		The default value is 4. A value of 0 disables pretty mode.
